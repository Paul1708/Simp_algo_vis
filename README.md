# AlgoViz

A simple browser-based sorting algorithm visualizer built with vanilla HTML, CSS and JavaScript.
<img width="951" height="538" alt="image" src="https://github.com/user-attachments/assets/d137b3c4-2fde-4306-bf92-12d8f2caa23c" />

## Features

- Generate random arrays
- Visualize Bubble Sort and Selection Sort
- Control speed, pause/resume, and reset

## Usage

Open `index.html` in a web browser. Use the controls to pick an algorithm, adjust speed, and start or pause the animation.
